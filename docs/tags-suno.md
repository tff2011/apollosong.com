Aqui está um guia prático em pt-BR (com exemplos) para usar tags do Suno de modo que o resultado permaneça emocionante, cantável e “correto” para a geração.

1) Regras de ouro de tags (pra não quebrar a estrutura)
Tags de seção (sempre em colchetes)

Use uma seção por bloco, assim:

[Intro]

[Verse]

[Pre-Chorus]

[Chorus]

[Bridge]

[Outro]

Não use barras antes do nome (tipo /Chorus).
Não invente nomes confusos. Se quiser variação, use:

[Chorus 2], [Verse 2], [Chorus Final]

Começo e fim (muito importante)

Coloque:

[start] no topo, sozinho

[end] no fim, sozinho

Sem letra dentro dessas tags.

2) Como dar direção musical sem bagunçar a letra
Direções musicais vão em colchetes, em inglês

Essas direções ajudam o Suno a entender emoção e performance. Exemplos bons:

[Soft piano intro, intimate vocal]

[Instrumental buildup: pads rising, kick enters]

[Big chorus lift: wide drums, bright synths, vocal harmonies]

[Half-time feel, darker tone]

[Drop: punchy bass, tight drums]

✅ Só direção musical dentro desses colchetes (sem frases de letra misturadas).

Backing vocals / harmonias vão em parênteses

Use parênteses apenas para partes de apoio/coro:

(oh-oh)

(stay with me)

(eu volto, eu volto) ← pode ser pt-BR aqui, porque é voz de apoio (letra)

3) O “mapa emocional” que faz soar emocionante (sem ser novelão)

Pensa em cada seção como uma função emocional:

[Verse] = cena + verdade (concreto, íntimo)

[Pre-Chorus] = tensão + promessa (cresce, acelera a emoção)

[Chorus] = catarse + frase-grude (poucas palavras, repetição)

[Bridge] = virada (confissão, mudança de perspectiva, “antes/depois”)

[Outro] = aceitação, despedida, eco do hook

Dica prática: emoção vem de contraste.
Verso contido → pré cresce → refrão explode.

4) Como escrever as linhas pra ficar “cantável” no Suno
Tamanho de linha (melhor pro canto)

Evite linhas longas demais (tipo 18–22 palavras).

Prefira 8–12 palavras por linha no verso.

No refrão, linhas curtas ganham (4–9 palavras) e repetem.

Refrão que gruda (modelo)

No [Chorus], faça:

Linha 1 = hook (título)

Linha 2 = imagem simples

Linha 3 = hook repetido (igual ou 1 palavra muda)

Linha 4 = decisão/promessa

5) Estruturas que “funcionam” pra emoção + viral

Se você quer emoção com impacto rápido:

Opção A (clássica e segura)
Intro → Verse → Pre → Chorus → Verse → Pre → Chorus → Bridge → Chorus Final → Outro

Opção B (chega no refrão cedo)
Intro → Chorus → Verse → Pre → Chorus → Bridge → Chorus Final → Outro

[Outro]
[Instrumental fade, vocal airy]
Eu deixo a luz acesa pra você

(3) Em branco (pode)
